# 🏥 SmartCare Hospital ERP System

A Hospital Management System developed using Flask, MySQL, HTML, CSS, and Python.

## 🚀 Features

* Patient Management

  * Add Patient
  * View Patient
  * Search Patient
  * Update Patient
  * Delete Patient

* Doctor Management

  * Add Doctor
  * View Doctor
  * Search Doctor
  * Update Doctor
  * Delete Doctor

* Appointment Management

  * Book Appointment
  * View Appointments
  * Update Status

* Medicine Management

  * Add Medicines
  * View Medicines
  * Update Medicines
  * Delete Medicines

* Billing Management

  * Generate Bills
  * View Bills

* Medical Reports

  * Add Reports
  * View Reports

* Feedback Management

  * Add Feedback
  * View Feedback

* Statistics Dashboard

  * Total Patients
  * Total Doctors
  * Total Appointments
  * Total Medicines
  * Total Revenue

## 🛠 Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* Jinja2

## ▶️ How to Run

1. Create MySQL database: `hospitaldb`
2. Run SQL tables
3. Install dependencies:

pip install flask mysql-connector-python

4. Run project:

python app.py

5. Open:

http://127.0.0.1:5000
